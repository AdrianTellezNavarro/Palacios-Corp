Proceso de Naturalización por Latinoamericano

Este es el proceso para la naturalización de personas latinoamericanas.

Requisitos

  • Ser ciudadano de un país latinoamericano.
  • Residencia mínima en el país.
  • Conocer la lengua española.
  • No tener antecedentes penales.

Documentación necesaria

  1. Pasaporte vigente.
  2. Acta de nacimiento.
  3. Comprobante de residencia.
  4. Fotografias tamaño infantil
  5. Carta de antecedentes no penales.

Pasos a seguir

Para iniciar el proceso, por favor dirígete a la oficina de migración más cercana o visita nuestro sitio web para más información.
